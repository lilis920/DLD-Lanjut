Tugas Neural Style Transfer - Generative Deep Learning

Repositori ini berisi tugas individu untuk mata kuliah Deep Learning Lanjut (Pertemuan 2). Proyek ini mengimplementasikan teknik Neural Style Transfer (NST) menggunakan TensorFlow dan Keras.

Tugas ini bertujuan untuk menggabungkan dua gambar:

Gambar Utama (Content): Objek foto asli(mabuci.jpg).

Gambar Gaya (Style): Lukisan atau pola artistik(lukisan.jpg).

Model akan menghasilkan gambar baru yang mempertahankan konten dari gambar utama namun dilukis dengan gaya dari gambar style.
